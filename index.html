Proyecto 2

En este proyecto teníamos que realizar un documento de HTML y que el nombre de la página apareciera en la ventana del navegador, era un requisito hacerlo y también teniamos que usar un enlace a una página ya designada.

Proyecto 3

Para este proyecto teníamos que realizar un formulario, debia contar con : Título, un input tipo text para el nombre y apellido un input tipo radio para elegir el genero, otro campo de texto tipo correo para el mismo, 2 casillas de verificación y un boton de enviar. Sin estilo.

Proyecto 4

Este proyecto consistía en crear 2 página que tuvieran estilo que quisieramos, era de libre elección y estubieran enlazadas entre sí, que la primera te llevara a la segunda y viceversa.

Proyecto 5

En este último proyecto de muestra consistía sobre una conjunción de los otros proyectos anteriores, porque es una página HTML, que contiene una hojas de estilo (CSS) externa, Que contiene un formulario y a su vez también contiene enlaces externos
